# CareerPilot AI

AI-powered career guidance platform that analyzes resumes and generates career roadmaps.

## Features

* User Authentication
* Resume Analysis
* Career Roadmap Generation
* Resume History
* User Profile Dashboard

## Tech Stack

### Frontend

* Next.js
* TypeScript
* Axios

### Backend

* Express.js
* TypeScript
* JWT Authentication

### Database

* PostgreSQL
* Prisma ORM

## Run Locally

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

* /dashboard
